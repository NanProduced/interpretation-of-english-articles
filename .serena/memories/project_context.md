# 项目上下文

## 简述

本仓库用于开发一个面向非零基础英语用户的微信小程序，核心能力是通过结构化 AI Workflow 对英文文章进行解读。

## 核心输出

- 词汇标注
- 语法分析
- 长难句拆解
- 高质量翻译

## 产品形态

- 轻量工具，而不是完整英语学习 App
- 采用前后端分离架构
- 早期阶段输出 Schema 稳定性优先于功能数量

## 计划技术栈

- 前端：Taro + React + TypeScript
- 后端：FastAPI + LangGraph + PydanticAI
- 后续：待基础 Workflow 稳定后再引入 RAG

## 当前优先事项

1. 定义输出 Schema
2. 搭建最小 Workflow
3. 打通小程序输入到后端
4. 稳定质量与渲染体验
